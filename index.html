BadmInTown


Welcome to BadmInTown, a Data Visualisation project using Leaflet.

This project displays data about badminton clubs and tournaments taking place in France.

How to use

You can click on department to gain focus on them and see the data points in details. Clicking on a data point yields supplementary information. If two or more data points have the exact same GPS coordinates, they are stacked in a cluster. You can open the cluster and this its points by clicking on it.

By default the map displays information about clubs, but you can toggle this by clicking on the (clubs) or (tournaments) buttons.

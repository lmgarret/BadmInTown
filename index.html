A responsive sidebar for the mapping library Leaflet.

Compatible with version 0.7 and 1.x (tested up to 1.2.0)

More examples

  • Autopanning of map
  • Right aligned
  • Dynamic panel content using the JS API
  • Proof that it works with leaflet 0.7 ;)
  • The sidebar adapts to map container size
